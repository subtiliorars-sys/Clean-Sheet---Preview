# Clean Sheet — Public Concept

**Sanitized preview · 2026-07-13**

---

## One-liner

A roguelite where **you are the goalkeeper** — the only position that matters, the only
mistake that ends everything, and the only score that counts is a clean sheet.

---

## Why it's fresh

| Typical football game | Clean Sheet |
|-----------------------|-------------|
| Control the whole team | Control only the keeper |
| Win by scoring | Win by *not* conceding |
| GK is CPU or QTE | GK is the entire game |
| 90-minute sim | 15–45 second save loops |

---

## Recommended core loop (concept)

1. Striker approaches — read body-language cues
2. Choose dive direction — make the save (or don't)
3. Between waves — pick upgrade cards
4. Between rounds — face a **Boss Striker** with signature patterns
5. Run ends after 3 goals conceded; meta unlocks new keepers and upgrades
