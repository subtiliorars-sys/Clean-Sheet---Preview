# Clean Sheet — Preview

The goalkeeper-as-protagonist game — a clean-room original, Steam-first.


### First-run tip

Open [Play](https://subtiliorars-sys.github.io/Clean-Sheet---Preview/) for the public build. Wait for the boot scuff before diving - feints skip the dust tell. Esc pauses; M mutes. Private HQ gameplay lives in Clean-Sheet.

## What is Clean Sheet?

Every football game puts you in the striker's boots or the manager's chair. **Clean Sheet**
flips the script: you are the keeper, the last line. Read the striker, choose your dive,
and build a roguelite career of saves — not goals scored.

**Genre:** Sports roguelite / arcade  
**Platform target:** Steam (desktop first)  
**Engine direction:** Phaser 3 + TypeScript + Electron (house packaging path)

## Core fantasy

- **Last line pressure** — one mistake ends the run
- **Pattern reading** — predict the shot, don't just react
- **Roguelite of saves** — waves of strikers, upgrade cards, boss strikers
- **Clean sheet = perfect run** — success is goals *not* conceded

## IP boundary

This is a **clean-room original**. It does not use the "Guardian of Goal" name, code, or
art from the Ilerioluwa academy website — those belong to separate IP.
